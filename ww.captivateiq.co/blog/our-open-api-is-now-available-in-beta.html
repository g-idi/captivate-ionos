Our Open API is Now Available in Beta

August 3, 2021
August 12, 2021

We’re excited to announce the Beta release of CaptivateIQ’s Open API. Customers can now extend the integration capabilities of their commissions' infrastructure by leveraging custom integrations and automations to streamline the sales incentives process.

Ecosystems

There’s a long list of tools that modern businesses use to scale and run their financial operations:

  • Customer Relationship Management (Salesforce)
  • Enterprise Resource Planning (i.e. NetSuite, Oracle)
  • Human Capital Management (i.e. ADP, Workday) 
  • Spend / Expense Management (i.e. QuickBooks, SAP Concur)
  • Financial Management (i.e. FinancialForce, Zuora)
  • Payroll & Workforce Management (i.e. Sage Intact)
  • Reporting & Analytics (i.e. Snowflake, MySQL)

With such a broad array of solutions, having a sales compensation tool that fits into this financial ecosystem is critical. CaptivateIQ’s new Open API gives customers the customization and automation they need to streamline their commissions process.

Syncing Data

When it comes to data inputs, sources can be a long unruly mess. Not all solutions have available connectors and those that do, don’t always make the data accessible. With the new data workbooks and worksheets endpoints, customers can build integrations to pull data from any source of their choosing and programmatically control sync frequencies. Commission admins will no longer need a technical resource to help them retrieve data from 3rd party sources. The Open API syncs data directly to the designated worksheet.

HCM Automation

Accessing source calculation is only half the battle of incentive management. Over the course of a pay period, so many things can happen:

  • New team members join a sales team
  • Internal promotions or team changes
  • Changes to team hierarchy and management
  • Changes to sales compensation input variables

CaptivateIQ’s Open API gives customers programmatic access to app components that control these vital inputs by letting the Admin users:

  • Control the creation of new employees or remove old ones
  • Change who manages designated teams and team hierarchy
  • Change which managers have access to team performance
  • Change which employee plan designation
  • Change variables used in plan calculations

The Open API gives customers complete control over automating changes to sales compensation and incentive management based on changes to source systems. Other benefits of the Open API include:

  1. Low cost
  2. Cloud-based which makes it available anywhere with an internet connection
  3. Enhances performance and productivity
  4. Truly integrated workflows that can be set up through automations
  5. Easy implementation that requires little to no code
  6. Greater functionality powered by the various integrations available


Commissions Extraction

Commissions data and calculations can be critical sources of data. Not just for the purpose of payroll processing but also in the context of analytics and compliance. CaptivateIQ’s Open API gives customers access to processed commissions payouts as well as their corresponding adjustments and calculations. Customers can automate commissions payout extraction to payroll systems or programmatically extract and store commissions payouts and all the corresponding calculations to a data warehouse. 

CaptivateIQ’s Open API gives customers a new level of customization and automation. For customers who don’t have access to build custom integrations or automations, our team can support these implementations on your behalf. Please reach out to us to learn more.
